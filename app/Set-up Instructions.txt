---- To activate virtual session ----
Make sure you are in the folder <app>
Windows:
    py -3 -m venv .venv
    .venv\Scripts\activate
    npm install
    pip install -r requirements.txt

Linux/Mac
    python3 -m venv .venv
    . .venv/bin/activate
    npm install
    pip install -r requirements.txt

---- Start Server ----
Windows:
    yarn startWin
or
    waitress-serve --listen=:5000 main:app
    

Linux/Mac:
    yarn startUni
or
    gunicorn -b :5000 main:app
